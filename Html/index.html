About Me

🙋 소개

안녕하세요! 사용 경험을 진지하게 고민하는 프론트엔드 개발자 김수종입니다.

단순히 작동하는 코드를 넘어, 실제로 사용할 때 느껴지는 경험의 질을 함께 설계합니다.

2D 게임과 인터랙티브 웹에서 영감을 받아, 재미있고 직관적인 인터페이스 만드는 것을 좋아합니다.

⚡ 핵심 가치

  • 사용성과 심미성을 동시에 추구하는 개발
  • 애니메이션과 인터랙션으로 생동감 있는 UI 구현
  • 반응형 웹 개발에 대한 높은 관심과 열정
  • 게임적 요소를 통한 독창적 사용자 경험 제공
  • 꾸준한 학습과 성장을 즐기는 개발자
